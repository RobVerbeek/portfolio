Containerised Apache webserver

The goal of this project was to learn the power of Docker. To do so I had to research how Docker worked. I set up an Apache webserver that communicates with an SQL database, furthermore to prove my mastery of Docker I decided to add a Nginx loadbalancer to this project.

During this project I learned various new technologies.

  • Creating images with dockerfiles
  • Containerizing an application
  • Making a Docker stack with Docker-compose
  • Horizontal web-scaling with Nginx
  • Ensuring data persistence
