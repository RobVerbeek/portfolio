Feathers & Fuselages

Wildlife strikes can be a serious issue, over the last 33 years there have been many incidents because of wildlife strikes. I used Qlik Sense to visualize the wildlife strikes on airplanes in the US. The goal of the project was to showcase the impact they can have on not only the economy but also the passengers aboard the airplanes. This project was part of the data visualization course during my studies.

Utopia Project

Voice-Over was done by a different Team Member

Utopia is a website created to help fresh graduates decide on what course they would like to pursue in higher education. This project was handed out to multiple groups which each had to come up with an interesting way of helping the graduates decide. My team wanted to create a game where the player would have to control the character and complete various objectives by writing code. My role was to handle the visuals. I created the maps in which the player moved in by coding, rendering and drawing the maps with asset sets.

Utopia Project

This docker stack was made for the Linux Webservices course I followed. To complete the project a containerized web page had to be setup. This web page runs on Apache which communicates with an SQL database. These services are containerized in Docker. The Apache container will communicate with the SQL container to retrieve the stored username and display it on the web page. As a bonus task the apache container had to be duplicated and loadbalanced, which I also managed to complete.

CompeteMore

CompeteMore is a PHP based website which aims to give the students of Thomas More an easy-to-use platform for hosting and participating in competitions. To create this website I teamed up with fellow students. Firstly the design stage, where we had to talk to the client. Analyze their needs and design prototypes.

The development of this project is still ongoing.
