Projects

Feathers and Fuselages

Wildlife strikes can be a serious issue, over the last 33 years there have been many incidents because of wildlife strikes. I visualized the recorded data from these wildlife strikes on airplanes in the US into several sheets of graphs. The goal of the project was to showcase the impact they can have on not only the economy but also the passengers aboard the airplanes. This project was part of the data visualization course during my studies.

Utopia

Utopia is a website created to help fresh graduates decide on what course they would like to pursue in higher education. This was a group project in which we chose to make a game where the player would have to control the character and complete various objectives by writing code. My role was to handle the visuals. I created the maps in which the player moved in by coding, rendering and drawing the maps with asset sets.
